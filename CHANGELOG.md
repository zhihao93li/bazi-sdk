## [1.0.0] - 2026-01-27

### 🎉 首次发布

#### 新增功能
- ✨ 核心八字计算引擎 (`BaziCalculator`)
- ✨ 完整的数据模型 (17 个 DTO 类)
- ✨ 真太阳时计算支持
- ✨ 四柱排盘 (天干地支、纳音、藏干、空亡)
- ✨ 日主强弱分析 (得令、得地、天干帮扶)
- ✨ 五行统计分析 (分布、喜忌)
- ✨ 十神计算
- ✨ 格局判断
- ✨ 大运流年计算 (10 步大运)
- ✨ 神煞计算
- ✨ Spring Boot 自动配置支持
- ✨ GitHub Packages 自动发布
- ✨ GitHub Actions CI/CD

#### 技术特性
- 📦 Maven 多模块架构
- 📦 Java 21 支持
- 📦 纯 Java Core 模块 (无 Spring 依赖)
- 📦 Spring Boot Starter 模块
- 📦 lunar-java 1.7.7 集成
- 📦 支持公历/农历输入
- 📦 支持年份范围: 1901-2100

#### 文档
- 📖 完整的 README.md
- 📖 GitHub Packages 发布指南
- 📖 5 分钟快速开始指南
- 📖 发布检查清单
- 📖 本地安装说明

#### 测试
- ✅ 15 个基础单元测试
